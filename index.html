1)Что такое let’s incrypt, для чего его используют?

Let’s Encrypt — это бесплатный, автоматизированный и открытый Центр сертификации (ЦС),
работающий на благо общества.

2)Утилита sertbot -для чего нужна?.

Certbot — клиент для автоматического создания и установки SSL-сертификата.

3)Имеет ли срок действия сертификат?

Да, иммеют свои сроки

Срок действия SSL-сертификатов истекает, он не длится вечно. Центр сертификации / Форум браузеров, который де-факто выступает в качестве регулирующего органа для индустрии SSL, заявляет, что срок действия SSL-сертификатов не должен превышать 27 месяцев. По сути, это означает, что SSL-сертификат можно использовать в течение двух лет, плюс до трех месяцев на продление срока действия предыдущего сертификата.

Цельурока:Основы git, branches, commits
Процесс выполнения: Инициализируйте в папке с сайтом из предыдущего урока гит репозиторий, сделайте коммит файла index.html, добавьте пару картинок и сделайте еще коммит.

1)Какая команда позволяет инициализировать проект в git?

Команда git init создает новый репозиторий Git. С ее помощью можно преобразовать существующий проект без управления версиями в репозиторий Git или инициализировать новый пустой репозиторий.

Как произвести коммит файла и для чего это вообще нужно?

Git commit - это команда для записи индексированных изменений в репозиторий Git. Прежде чем создавать очередной коммит, необходимо проиндексировать файлы в рабочей области с помощью команды git-add. Новый коммит будет включать текущие состояния индексированных файлов плюс последние сохраненные состояния неиндексированных (но отслеживаемых) файлов.

3)Как добавить все файлы в отслеживание git?

Git add * Команда git add добавляет изменение из рабочего каталога в раздел проиндексированных файлов. Она сообщает Git, что вы хотите включить изменения в конкретном файле в следующий коммит
